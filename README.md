 Machine Learning Add-on Course

OVERVIEW
This repository contains the hands-on tasks and notebooks completed during a **5-Day Machine Learning Add-on Course**. The course focused on building practical skills in Python, Image Processing, Machine Learning, Deep Learning, and Convolutional Neural Networks (CNN).

Day 1 – Scientific Python
- Introduction to Python for Data Science
- NumPy
- Pandas
- Data Manipulation

Day 2 – Image Processing
- Image Reading and Processing
- Basic Image Operations
- OpenCV Fundamentals

Day 3 – Machine Learning
- Data Preprocessing
- Machine Learning Concepts
- Model Training and Evaluation

Day 4 – Deep Learning & CNN
- Introduction to Deep Learning
- Neural Networks
- CNN Implementation
- Brain Image Classification Task
